# Cola_dream

![1](README_img/1.png)



[EasyX是针对 C++ 的图形库]: https://easyx.cn/
[b站演示视频]: https://www.bilibili.com/video/av25043844



如果想要运行可以直接clone exe文件夹，里面有可运行文件
